Este programa é um simulador de Máquina de Turing com Fita Limitada.
Faz a leitura das especificações de uma máquina em arquivo e permite realizar testes de palavras.


-- INSTRUÇÕES --

Iniciando o programa, será exibido o menu de escolhas:

    0 - Sair
    1 - Testar cadeia de palavras
    Digite a opcao desejada: 

Para sair do programa, digite 0.
Para testar uma palavra, digite 1 e, após, digite o nome do arquivo.extensão com as especificações
da máquina.

    Digite o nome do arquivo: 

Em seguida, digite a cadeia de símbolos que desejas testar, respeitando o alfabeto adicionado.

    Digite a cadeia de palavras: 

O resultado da cadeia digitada será exibido em seguida.
Exemplo:

    Digite a cadeia de palavras: aba

    *[q0]aba$
    *X[q1]ba$
    *[q1]XYa$
    REIJEITA

Continue testando mais cadeias ou encerre o programa digitando 0.
